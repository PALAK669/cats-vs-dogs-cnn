# Cats vs Dogs Classification using CNN

A Convolutional Neural Network (CNN) built using TensorFlow/Keras to classify images as cats and dogs.

## Technologies
- Python
- TensorFlow
- Keras
- NumPy
- Matplotlib

## Model Architecture
- Conv2D(32)
- MaxPooling2D
- Conv2D(64)
- MaxPooling2D
- Flatten
- Dense(128)
- Dense(1, Sigmoid)

## Results
- Test Accuracy: 87.5%
